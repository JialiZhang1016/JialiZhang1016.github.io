MS-YOLO: Infrared Object Detection for Edge Deployment

2024–2025

+3% mAP with 1.5% lower latency — real-time infrared detection on edge devices

Key Contributions

  • Designed a lightweight object detection model for thermal/infrared urban imagery, specifically engineered for deployment on edge devices with limited compute.
  • Proposed a novel SlideLoss function to mathematically re-weight gradients during backpropagation, effectively solving the severe class imbalance inherent in thermal datasets.
  • Replaced the standard YOLO backbone with MobileNetV4, significantly reducing parameter count while maintaining feature extraction capability for low-contrast infrared images.
  • Achieved +3% mAP (Mean Average Precision) over state-of-the-art baselines and 1.5% latency reduction on the FLIR dataset, viable for real-time surveillance.
  • Part of a $475K DEVCOM-funded research project on AI-enhanced infrared analysis (PI: Prof. Donald C. Wunsch II).
