Motivation

Problems with Recurrent Models

  • Sequential computation prevents parallelization — each hidden state depends on the previous one, making training slow on modern hardware
  • Vanishing gradients over long sequences — information from early tokens is progressively lost as the sequence grows
  • Limited context window — practical memory constraints restrict the effective receptive field of RNNs and LSTMs

Key insight: Attention mechanisms can replace recurrence entirely, enabling full parallelization.

Paradigm Comparison

Sequence Modeling Approaches

RNN

Sequential processing of tokens one at a time. Path length between positions is O(n) steps. Limited context due to vanishing gradients and fixed-size hidden state.

LSTM

Gated memory cells improve gradient flow and long-range dependencies. However, computation remains strictly sequential, limiting training throughput.

Transformer

Fully parallel computation across all positions. O(1) sequential operations with global context — every token attends to every other token directly.
