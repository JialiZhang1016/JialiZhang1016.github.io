LLM-based Retrieval-Ranking Pipeline

Completed | Sep 2025

Kaggle Top 9% (114/1282) — Bronze Medal with only 3 submissions

Key Contributions

  • Architected a “Recall-then-Precision” pipeline: high-speed regex for initial candidate retrieval, followed by a 32B-parameter LLM (Qwen/Llama-3) for semantic classification.
  • Optimized inference using vLLM and context-window management, enabling processing of thousands of papers within strict competition compute constraints.
  • Designed custom pre/post-processing filters based on statistical error analysis of training data, reducing LLM hallucinations and false positives.
  • Achieved Bronze Medal with only 3 submissions (vs. 100+ by competitors) — demonstrating a data-centric engineering approach over brute-force iteration.
